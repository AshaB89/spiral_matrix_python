# Spiral Matrix in Python

This repository contains a Python implementation to generate an **N × N spiral matrix**.

## Problem

Create a square matrix filled with numbers from 1 to N² arranged in a spiral order.

Example for n = 3
```
1 2 3
8 9 4
7 6 5
```
## Features

* Generates spiral matrix
* Prints matrix as a 2D list
* Prints matrix in formatted grid form

## Language

Python
